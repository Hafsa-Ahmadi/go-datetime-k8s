Allow users to refine scan results using advanced filters.

## Available Filters

- **By Scan Status:**

  - Start and End date pickers

- **By Image Name/Tag:**
  - Search bar with autocomplete

## UX Notes
- Filters applied appear as tags above the results
- Clear All button resets filters
- Real-time filtering with debounce
